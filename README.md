## 🚀 Características de Élite

### 📊 Editor de Tablas Profesional (Tipo Word/Excel)
- **Fusión y División**: Combina celdas vertical u horizontalmente con un clic.
- **Redimensionamiento Manual**: Ajusta el ancho de columnas y alto de filas arrastrando los bordes con el mouse.
- **Gestión Estructural**: Añade o elimina filas y columnas manteniendo la integridad del diseño.
- **Soporte Multimedia**: Inserta imágenes directamente en las celdas de la tabla.

### 🔄 Previsualización de Datos en Tiempo Real (Live Preview)
- **Interpolación Dinámica**: Escribe etiquetas como `{{nombre}}` o `{{total}}` y visualiza instantáneamente los datos de muestra.
- **Modo de Edición Inteligente**: Al editar, ves las etiquetas originales; al terminar, ves los datos reales.
- **Explorador de Datos**: Panel lateral para gestionar y copiar variables del dataset con facilidad.

### 🎨 Motor de Diseño Avanzado
- **Capas (Layers)**: Gestiona el orden Z de los elementos desde un panel dedicado.
- **Propiedades Premium**: Control total sobre tipografía (Google Fonts), sombras, bordes, rellenos y alineaciones verticales.
- **Glassmorphism UI**: Una interfaz diseñada para impresionar, con efectos de desenfoque y transiciones suaves.
- **Flexibilidad**: Soporte para cuadros de texto, imágenes, formas y gráficos.

### 📄 Exportación de Alta Calidad
- Diseñado para generar PDFs impecables manteniendo cada detalle de espaciado y diseño.

## 🛠️ Stack Tecnológico

- **Frontend**: [React.js](https://reactjs.org/)

## 📦 Instalación

1. Clona el repositorio:
   ```bash
   git clone https://github.com/tu-usuario/proyectoFactura.git
   ```

2. Instala las dependencias:
   ```bash
   npm install
   ```

3. Inicia el servidor de desarrollo:
   ```bash
   npm run dev
   ```

---

## 💡 Cómo usar el Diseñador

1. **Añadir Elementos**: Usa la barra superior para insertar textos, tablas o imágenes.
2. **Personalizar**: Selecciona cualquier elemento para abrir el panel de **AJUSTES** lateral. Allí podrás cambiar colores, bordes, sombras y fuentes.
3. **Tablas Mágicas**:
   - Selecciona múltiples celdas (Shift + Click) para unirlas.
   - Pasa el mouse por los bordes internos para redimensionar con el cursor de flechas.
4. **Vincular Datos**: Abre la pestaña **DATOS** en el lateral, copia una etiqueta y pégala en un texto. ¡Verás cómo se reemplaza automáticamente!
